PROJECT DESCRIPTION

This project is a responsive landing page developed as part of the Lawctopus Frontend Development assignment.

The landing page is idesigned based on the provided assignment reference while implementing a fresh responsive UI using Bootstrap and modern frontend practices. but redesigned using modern UI principles, improved spacing, responsive layouts, reusable components, and subtle animations while maintaining the original content structure.

HOW TO RUN

1. Extract the project folder.
2. Open the folder in Visual Studio Code.
3. Install the Live Server extension (if not installed).
4. Right-click index.html.
5. Select "Open with Live Server".

Alternatively, simply open index.html in any modern web browser.
